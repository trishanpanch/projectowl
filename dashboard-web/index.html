Get Started

  1. Download the Extension

    Currently in Beta. Download the source code to install manually.

  2. Load into Chrome

    1. Open Chrome and go to chrome://extensions
    2. Enable "Developer mode" in the top right.
    3. Click "Load unpacked" and select the folder you just downloaded.

  3. Start work

    Click the O icon in your browser toolbar to link it to your parent account.

  4. Connect to Dashboard

    Connect to your web dashboard to see what your child has seen and receive structured feedback on how to discuss with your child and build digital resilience
